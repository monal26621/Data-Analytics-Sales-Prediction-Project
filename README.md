# Data-Analytics-Sales-Prediction-Project
This project focuses on data analytics, visualization, and predictive modeling using Python. The notebook performs:  Data collection and understanding Data cleaning &amp; preprocessing Exploratory Data Analysis (EDA) Data visualization Machine Learning prediction using Linear Regression.

**Features**

✅ Data preprocessing and cleaning
✅ Duplicate and missing value handling
✅ Sales and order analysis
✅ Revenue insights
✅ Interactive visualizations using Matplotlib
✅ Predictive analytics using Linear Regression
✅ Model evaluation with regression metrics
